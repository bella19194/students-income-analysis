# Case 7 — Sales Data Quality Pipeline

## Описание
Пайплайн для обработки и контроля качества данных о продажах.
Объединяет несколько CSV-файлов, очищает данные, считает процент брака
и формирует итоговый отчёт по выручке.

## Структура проекта
- data/raw/ — исходные CSV-файлы
- data/clean/ — очищенные данные
- data/reports/ — итоговые отчёты
- logs/ — лог выполнения пайплайна
- src/pipeline.py — основной код

## Логика пайплайна
1. Чтение всех CSV из `data/raw`
2. Объединение в одну таблицу
3. Приведение типов данных
4. Проверка качества данных
5. Отделение валидных и невалидных строк
6. Подсчёт процента брака
7. Расчёт выручки
8. Формирование отчёта по продуктам

## Результаты
- `sales_clean.csv` — очищенные данные
- `monthly_report.csv` — отчёт по выручке
- `pipeline.log` — лог выполнения

## Запуск
```bash
python src/pipeline.py
